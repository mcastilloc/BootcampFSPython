Proyecto: poo_sistema_educacional

Descripción:
Implementación de un sistema educativo usando Programación Orientada a Objetos en Python.
Incluye clases: Persona, Alumno, Profesor, Asignatura, Curso.

Conceptos aplicados:
- Herencia (Alumno y Profesor heredan de Persona)
- Sobrecarga de métodos (registrar nota admite un valor único o lista)
- Polimorfismo (mostrar_info sobreescrito en Alumno y Profesor)
- Interacción entre objetos (asignaturas, cursos y alumnos relacionados)

Ejecución:
1. Asegúrese de tener Python 3.x instalado.
2. Ejecutar: python main.py
3. Se mostrará en consola la información de alumnos, profesor, asignatura y curso.
