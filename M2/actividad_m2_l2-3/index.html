Mi primer proyecto web

Este proyecto demuestra la estructura básica de un documento HTML con seis etiquetas diferentes.

  • <header> -> Encabezado de la página
  • <section> -> Contenido principal
  • <article> -> Bloque de contenido independiente
  • <aside> -> Contenido adicional o lateral
  • <footer> -> Pie de página
  • <p> -> Párrafo de texto
